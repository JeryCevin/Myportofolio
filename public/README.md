# Public Assets

## Foto Profil

Untuk menampilkan foto profil Anda di Hero section:

1. Simpan foto Anda dengan nama: `profile.jpg`
2. Letakkan di folder: `c:\xampp\htdocs\Portofolio\public\`
3. Gunakan foto yang sudah Anda kirimkan (foto dengan seragam biru-hitam dengan bendera Indonesia)

**Persyaratan foto:**
- Format: JPG atau PNG
- Nama file: `profile.jpg` (harus persis seperti ini)
- Ukuran yang disarankan: minimal 800x1000px untuk kualitas terbaik
- Posisi: Pastikan wajah berada di bagian atas foto

**Lokasi lengkap foto:**
```
c:\xampp\htdocs\Portofolio\public\profile.jpg
```

Setelah menyimpan foto, refresh browser untuk melihat perubahan!
